# Skup podataka o glazbenim instrumentima

Ovaj skup podataka sadrži detaljne informacije o različitim glazbenim instrumentima, uključujući njihove vrste, ključ, polifoniju, zemlju podrijetla, stoljeće nastanka, materijal, prosječnu težinu, podvrste, žanrove i poznate predstavnike. 
Podaci su organizirani tako da omogućuju analizu hijerarhijskih odnosa između instrumenata i njihovih povezanih karakteristika.

## Metapodaci

- **Naziv skupa podataka:** Glazbeni instrumenti
- **Autor:** Vinko Šapina
- **Verzija:** 1.0
- **Jezik podataka:** hrvatski
- **Format podataka:** CSV i JSON
- **Opis licence:** Creative Commons CC0 1.0 Universal
- **Datum stvaranja:** 26. listopada 2025.
- **Opis atributa:**
  - `nazivInstrumenta`: naziv instrumenta
  - `vrsta`: vrsta instrumenta (puhački, gudački, žičani...)
  - `kljuc`: glazbeni ključ instrumenta (G, F, C)
  - `polifonija`: monofon ili polifon (može proizvoditi jedan ili više tonova odjednom)
  - `zemljaPodrijetla`: zemlja u kojoj je instrument nastao
  - `stoljeceNastanka`: stoljeće u kojem je instrument okvirno nastao
  - `materijal`: materijal od kojeg je instrument izrađen
  - `prosjecnaTezinaKg`: prosječna težina instrumenta u kilogramima
  - `podvrste`: lista podvrsta instrumenta s rasponom tonova svake podvrste
  - `žanrovi`: glazbeni žanrovi u kojima se instrument koristi
  - `predstavnici`: poznati izvođači koji sviraju instrument
- **Namjena:** Analiza glazbenih instrumenata, edukacija i vizualizacija podataka
- **Izvor podataka:** Primarni podaci stvoreni za laboratorijske vježbe
- **Napomene:** Skup podataka je primjer strukturiranja relacijskih i hijerarhijskih podataka u CSV i JSON formatu.
